## Customization

### Colors
Modify CSS variables in `styles.css`:
```css
:root {
    --cosmic-purple: #6a0dad;
    --cosmic-blue: #00ced1;
    --cosmic-gold: #ffd700;
    /* ... more color variables */
}
```

### Frequencies
Update timeline frequencies in `portal.js`:
```javascript
frequencies: {
    past: 432,
    present: 528,
    future: 963
}
```

### Content
Edit modal content in the `openPortalModal()` function in `portal.js`.
